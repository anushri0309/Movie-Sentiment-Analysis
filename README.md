
# Movie-Sentiment-Analysis

An end-to-end movie review sentiment analysis web application built with Python, scikit-learn, and Streamlit.  
The app predicts whether a given movie review expresses a **positive** or **negative** sentiment.

## Features

- Pretrained machine learning model for binary sentiment classification on movie reviews.
- Text preprocessing pipeline including tokenization and vectorization.
- Simple, interactive Streamlit UI for entering custom reviews and viewing predictions.
- Reusable trained artifacts (`model.joblib` and `vectorizer.joblib`) for fast inference.
